## 二、已可用功能

### 2.1 AI 牌局摘要（结算页）

- **文件**：`miniprogram/utils/ai.js` → `generateAISummary()`
- **调用**：牌局结算时
- **配置**：`DEEPSEEK_API_KEY`（填入 `miniprogram/config/env.js`）
- **兜底**：不配置 Key 时使用内置 30+ 本地模板
- **状态**：✅ 可用

### 2.2 AI 数据分析（分析页）

- **文件**：`miniprogram/pages/analysis/analysis.js`
- **调用**：用户打开 AI 分析 Tab
- **配置**：无，纯本地计算
- **限制**：完整报告需通过激励视频广告解锁
- **状态**：⚠️ 部分可用

### 2.3 Canvas 分享海报

- **文件**：`miniprogram/utils/poster.js`
- **调用**：结算页点击分享
- **配置**：无
- **状态**：✅ 可用

- **文件**：`miniprogram/pages/room/room.js` → `announceJoin()`
- **接口**：预留，未实际接入
- **需要**：微信同声传译插件（`wx069ba97219f66d99`）
- **状态**：❌ 预留接口，需手动接入

如需接入：
1. 在微信公众平台后台添加「微信同声传译」插件
2. 在 `app.json` 中声明插件（见下方代码）
3. 在 `announceJoin()` 中完善调用

```javascript
// app.json
{
  "plugins": {
    "WechatSI": {
      "version": "0.3.6",
      "provider": "wx069ba97219f66d99"
    }
  }
}
```

## 四、广告功能

广告占位符已存在于代码中，但需要先开通微信流量主（UV >= 500）才能填入真实广告位 ID。

| 广告类型 | 位置 | 配置项 |
|----------|------|--------|
| Banner | 个人中心底部 | `AD_BANNER_ID` |
| 激励视频 | 分析页解锁完整报告 | `AD_REWARDED_VIDEO_ID` |

---

## 五、推荐的接入顺序

### 现在就可以用
1. ✅ 配置 AppID（必做）
2. ✅ 配置 DeepSeek API Key（可选，有本地模板兜底）

### 有用户量之后
3. 开通云开发（如果需要多端数据）
4. 部署云函数

### UV >= 500 后
5. 开通流量主
6. 填入广告位 ID

### 可选（需手动接入）
7. 添加微信同声传译插件（语音播报）
