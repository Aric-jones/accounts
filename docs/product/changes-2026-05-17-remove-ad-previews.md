# 取消广告位预览

## 变更级别

L1：跨多个页面的展示和局部交互调整；不改变 `transactions` 主数据模型，不新增云函数契约。

## 变更目的

当前广告只属于预留能力，尚未确定是否接入微信流量主。为避免用户误以为已经存在真实广告或广告解锁权益，本次取消所有页面中的模拟广告位预览，并暂停分析页的激励广告解锁入口。

## 规格说明

- 首页、开桌页、房间页、历史页、AI 分析页、结算页、个人中心页不展示模拟 Banner、信息流、激励视频广告块。
- AI 分析页不再调用 `wx.createRewardedVideoAd`，完整报告状态默认开放，不以广告观看作为解锁条件。
- `AD_BANNER_ID`、`AD_REWARDED_VIDEO_ID` 暂不作为当前运行配置项描述；后续重新接入广告时需另行补规格与验收。
- `components/ad-banner` 暂保留为未消费的兼容组件，不在当前页面主路径中使用。

## 验收点

- 全仓库页面 WXML 中不再出现 `ad-mock`、`ad-zone` 模拟广告容器。
- AI 分析页不再出现“观看广告解锁”按钮，不再调用激励视频广告 API。
- 配置文档和 README 不再描述广告位 ID 为当前需要配置的能力。
- 创建房间、加入房间、记账、结算、历史、分析页面的核心入口仍可打开。
